# NumPy 数组操作练习

本练习旨在帮助你熟悉 NumPy 数组的创建和操作，以及在特定条件下进行数值计算。

## 练习内容

### 1. 幂运算相关练习

#### 1.1 查找特定幂的末位数字
实现 `find_power_with_six()` 函数，找出所有满足以下条件的整数 i：
- 0 < i < 100
- 2^i 的最后一位数字是6

#### 1.2 计算2的幂
实现 `first_ten_powers()` 函数，计算并返回2的前10次幂（从0次幂到9次幂）。
